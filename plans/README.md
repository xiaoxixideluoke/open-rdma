# Open-RDMA 学习计划

> 学习目录，按天组织，每天一个文件。

| 文件 | 内容 |
|------|------|
| [WEEKLY_PLAN.md](WEEKLY_PLAN.md) | 本周总计划 + 7 天概览 |
| [DAY1.md](DAY1.md) | Day 1 — 环境搭建 + 第一个 RDMA 程序 |
| [DAY1_REPORT.md](DAY1_REPORT.md) | Day 1 详细学习报告（含 debug 记录、知识总结） |

每天完成任务后，修改文件中的 `- [ ]` 为 `- [x]`。

---

## 快速开始

```bash
# 查看本周计划
cat plans/WEEKLY_PLAN.md

# 开始今天的学习
cat plans/DAY1.md
```
